# Documentation for this Crypto Analysis API:

## Used Resources:

### 1. Coin Market Cap (API)

https://pro.coinmarketcap.com/account
and
https://coinmarketcap.com/api/documentation/v1/

Endpoints:

```
"COIN_CAP_BASE_URL": "https://pro-api.coinmarketcap.com/v1"
```
